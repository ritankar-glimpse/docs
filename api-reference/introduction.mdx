The Glimpse API exposes prediction market data through a set of REST endpoints under `/api/v1/nmarket`. These docs are designed to be copied directly into LLM context windows so builders can integrate Glimpse markets into their own apps and agents.

## Base URL

```text
https://api.glimpse.example.com
```

Replace with your actual production host.

## Conventions

- All monetary values are returned in **millisatoshis** (`millisats`). 1 sat = 1,000 millisats.
- Timestamps are Unix epoch seconds in UTC unless noted.
- `topic_id` is a numeric market identifier. `batch_id` is a UUID grouping related markets.

## Endpoint index

| Method | Path | Purpose |
| --- | --- | --- |
| GET | `/api/v1/nmarket/markets/:topic_id/stats` | Per-market volume \+ liquidity stats |
| GET | `/api/v1/nmarket/batches/:batch_id/stats` | Per-batch volume \+ liquidity stats |
| GET | `/api/v1/nmarket/markets/:topic_id/quotes` | Full market detail with quotes |
| GET | `/api/v1/nmarket/batches/:batch_id/active-markets` | All active markets in a batch |
| GET | `/api/v1/nmarket/v2/batches/:batch_id/active-markets` | v2 paginated active markets |
