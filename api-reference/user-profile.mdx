All User Profile endpoints require **User JWT**.

## Endpoints

| Method | Path | Auth | Purpose |
| --- | --- | --- | --- |
| GET | `/api/v1/loggedin-user-info` | User JWT | Caller's profile info. |
| POST | `/api/v1/update-profile` | User JWT | Update general profile fields. |
| POST | `/api/v1/update-mobile-number` | User JWT | Change mobile number. |
| POST | `/api/v1/confirm-mobile-number` | User JWT | Confirm mobile number via code. |
| POST | `/api/v1/update-bitcoin` | User JWT | Update saved BTC address. |
| POST | `/api/v1/update-lightning` | User JWT | Update saved Lightning address. |
