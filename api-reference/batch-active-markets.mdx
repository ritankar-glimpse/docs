---
title: "List active markets in a batch"
sidebarTitle: "Batch active markets"
description: "All active markets within a batch with cumulative stats"
---

```http
GET /api/v1/nmarket/batches/:batch_id/active-markets
```

Returns all active markets in a batch along with cumulative volume and liquidity. Use the [v2 endpoint](/api-reference/batch-active-markets-v2) when you need pagination.

## Path parameters

| Name | Type | Description |
| --- | --- | --- |
| `batch_id` | UUID | Identifier of the batch. |

## Response

```json
{
  "batch_id": "c6dd0be7-b9b8-4a8a-b735-623660c38982",
  "main_topic_title": "Weekly BTC Markets",
  "topic_count": 8,
  "count": 8,
  "cumulative_volume_millisats": 28500000,
  "cumulative_liquidity_millisats": 12000000,
  "markets": [
    {
      "topic_id": 12345,
      "title": "BTC above $100k?",
      "is_active": true,
      "is_resolved": false,
      "quote_mode": "active",
      "total_amount_in_market": 500000,
      "volume_24h_millisats": 1500000,
      "total_volume_millisats": 9800000,
      "outcomes": [ "..." ]
    }
  ]
}
```

### Top-level fields

| Field | Type | Description |
| --- | --- | --- |
| `batch_id` | UUID | Echoes the requested batch id. |
| `main_topic_title` | string | Display title for the batch. |
| `topic_count` | integer | Total topics in the batch. |
| `count` | integer | Number of markets returned. |
| `cumulative_volume_millisats` | integer | Sum of lifetime volume across markets. |
| `cumulative_liquidity_millisats` | integer | Sum of locked liquidity across markets. |
| `markets` | array | Per-market objects (see below). |

### Market object fields

| Field | Type | Description |
| --- | --- | --- |
| `topic_id` | integer | Market identifier. |
| `title` | string | Market question. |
| `is_active` | boolean | Whether the market is open for trading. |
| `is_resolved` | boolean | Whether the market has resolved. |
| `quote_mode` | string | Quote engine mode. |
| `total_amount_in_market` | integer | Total notional, millisats. |
| `volume_24h_millisats` | integer | 24h volume. |
| `total_volume_millisats` | integer | Lifetime volume. |
| `outcomes` | array | Outcome objects. |
