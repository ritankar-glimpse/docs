<Info>
  **Endpoint:** `GET /api/v1/nmarket/batches/{batch_id}/stats`\
  **Auth:** Public
</Info>

Returns aggregate trading and liquidity statistics for a batch.

## Parameters

### Path

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `batch_id` | string (UUID) | yes | Batch identifier. |

## Request

```bash
curl -X 'GET' \
  'https://main.bpmapi.io/api/v1/nmarket/batches/c6dd0be7-b9b8-4a8a-b735-623660c38982/stats' \
  -H 'accept: application/json'
```

## Server response

```json
{
  "batch_id": "c6dd0be7-b9b8-4a8a-b735-623660c38982",
  "volume_24h_millisats": 4200000,
  "total_volume_millisats": 28500000,
  "liquidity_locked_millisats": 12000000,
  "active_market_count": 8
}
```

## Responses

### `200 OK`

| Field | Type | Description |
| --- | --- | --- |
| `batch_id` | string (UUID) | Batch identifier. |
| `volume_24h_millisats` | integer | Volume traded in the last 24 hours across the batch, in millisats. |
| `total_volume_millisats` | integer | Lifetime trading volume across the batch, in millisats. |
| `liquidity_locked_millisats` | integer | Liquidity locked across the batch, in millisats. |
| `active_market_count` | integer | Number of currently active markets in the batch. |
