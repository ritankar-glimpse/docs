<Info>
  **Endpoint:** `GET /api/v1/nmarket/batches`\
  **Auth:** Public
</Info>

Returns the list of currently configured market batches.

## Parameters

None.

## Request

### cURL

```bash
curl -X 'GET' \
  'https://main.bpmapi.io/api/v1/nmarket/batches' \
  -H 'accept: application/json'
```

### Request URL

```text
https://main.bpmapi.io/api/v1/nmarket/batches
```

## Responses

### `200 OK`

**Schema**

```json
{
  "batches": [
    {
      "batch_id": "string",
      "end_time_utc": 0,
      "main_topic_title": "string",
      "outcome_interval": 0,
      "outcome_max_value": 0,
      "outcome_min_value": 0,
      "start_time_utc": 0,
      "topic_count": 0
    }
  ],
  "count": 0
}
```

**Fields**

| Field | Type | Description |
| --- | --- | --- |
| `batches[]` | array | List of batch objects. |
| `batches[].batch_id` | string (UUID) | Unique identifier for the batch. |
| `batches[].main_topic_title` | string | Display title for the batch. |
| `batches[].start_time_utc` | integer | Batch start time, Unix seconds UTC. |
| `batches[].end_time_utc` | integer | Batch end time, Unix seconds UTC. |
| `batches[].topic_count` | integer | Number of topics (markets) in the batch. |
| `batches[].outcome_min_value` | integer | Lower bound of outcome value range. |
| `batches[].outcome_max_value` | integer | Upper bound of outcome value range. |
| `batches[].outcome_interval` | integer | Step size between adjacent outcomes. |
| `count` | integer | Total number of batches returned. |
