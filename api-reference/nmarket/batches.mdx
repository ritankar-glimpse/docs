<Info>
  **Endpoint:** `GET /api/v1/nmarket/batches`\
  **Auth:** Public
</Info>

Returns the list of market batches configured on Glimpse.

## Request

```bash
curl -X 'GET' \
  'https://main.bpmapi.io/api/v1/nmarket/batches' \
  -H 'accept: application/json'
```

## Responses

### `200 OK`

Returns an array of batch objects.

| Field | Type | Description |
| --- | --- | --- |
| `batch_id` | string (UUID) | Batch identifier. |
| `main_topic_title` | string | Display title for the batch. |
| `topic_count` | integer | Number of topics in the batch. |
