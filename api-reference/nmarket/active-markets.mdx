<Info>
  **Endpoint:** `GET /api/v1/nmarket/batches/{batch_id}/active-markets`\
  **Auth:** User JWT (raw token in `Authorization` header, no `Bearer ` prefix)
</Info>

Returns active markets for the given batch, including outcomes, pricing, and aggregate stats.

## Parameters

### Path

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `batch_id` | string (UUID) | yes | Batch to fetch active markets for. |

## Request

```bash
curl -X 'GET' \
  'https://main.bpmapi.io/api/v1/nmarket/batches/c6dd0be7-b9b8-4a8a-b735-623660c38982/active-markets' \
  -H 'accept: application/json' \
  -H 'Authorization: <USER_JWT>'
```
