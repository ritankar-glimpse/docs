<Info>
  **Endpoint:** `POST /api/v1/nmarket/exit-consolidated`\
  **Auth:** Required
</Info>

Exits a consolidated position on a single topic for the authenticated user.

## Parameters

### Body

**Content type:** `application/json`

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `topic_id` | integer | yes | Market (topic) identifier. |
| `option_id` | integer | yes | Identifier of the option position to exit. |
| `shares` | integer | yes | Number of shares to exit from the consolidated position. |

## Request

```bash
curl -X 'POST' \
  'https://main.bpmapi.io/api/v1/nmarket/exit-consolidated' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "topic_id": 1944,
    "option_id": 101,
    "shares": 10
  }'
```
