<Info>
  **Endpoint:** `GET /api/v1/nmarket/ended-past-168h`\
  **Auth:** Public
</Info>

Returns ended markets for a batch within the past 168 hours (7 days).

## Parameters

### Query

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `batch_id` | string (UUID) | yes | Batch ID to fetch ended markets for. |

## Request

```bash
curl -X 'GET' \
  'https://main.bpmapi.io/api/v1/nmarket/ended-past-168h?batch_id=c6dd0be7-b9b8-4a8a-b735-623660c38982' \
  -H 'accept: application/json'
```

### Request URL

```text
https://main.bpmapi.io/api/v1/nmarket/ended-past-168h?batch_id=c6dd0be7-b9b8-4a8a-b735-623660c38982
```

## Responses

### `200 OK`

**Content type:** `application/json`

The OpenAPI schema for this endpoint is loosely typed (`additionalProp1: {}`), but the actual response is a `success` flag plus a `markets[]` array.

**Top-level fields**

| Field | Type | Description |
| --- | --- | --- |
| `success` | boolean | `true` on a successful response. |
| `markets[]` | array | Markets that ended within the past 168 hours for the requested batch. |

**Market object**

| Field | Type | Description |
| --- | --- | --- |
| `topic_id` | integer | Market identifier. |
| `title` | string | Market question / display title. |
| `description` | string | Long-form description. |
| `topic_type` | string | Topic type (e.g. `btc`). |
| `category` | string | Display category. |
| `batch_id` | string (UUID) | Parent batch id. |
| `end_time_utc` | integer | Market end time, Unix seconds UTC. |
| `estimated_resolve_time` | integer | Estimated resolution time, Unix seconds UTC. |
| `date_created` | integer | Creation timestamp, Unix seconds UTC. |
| `ended_at_utc` | integer | Actual end timestamp, Unix seconds UTC. |
| `num_outcomes` | integer | Total number of outcomes defined for the market. |
| `is_active` | boolean | Whether the market is open for trading. |
| `is_resolved` | boolean | Whether the market has resolved. |
| `resolved_option_id` | integer | `option_id` of the winning outcome (if resolved). |
| `market_subsidised` | boolean | Whether the market has subsidised liquidity. |
| `subsidy_amount` | integer | Subsidy amount, in millisats. |
| `total_amount_in_market` | integer | Total notional in the market, in millisats. |
| `resolution_info` | string | Plain-language description of the resolution rule. |
| `outcomes[]` | array | Outcome objects. |

**Outcome object**

| Field | Type | Description |
| --- | --- | --- |
| `option_id` | integer | Outcome identifier within the market. |
| `name` | string | Display label (e.g. price bin). |
| `yes_price` | number | YES price in sats. |
| `no_price` | number | NO price in sats. |
| `yes_price_millisats` | integer | YES price in millisats. |
| `no_price_millisats` | integer | NO price in millisats. |
| `shares` | number | Outstanding shares for this outcome. |
| `subsidised_shares` | number | Subsidised shares contributed to this outcome. |

{
  "additionalProp1": {}

### `500 Internal Server Error`
