<Info>
  **Endpoint:** `POST /api/v1/nmarket/exit-multi-topic-multi-leg`\
  **Auth:** Required
</Info>

Exits positions across multiple topics and legs in a single request for the authenticated user.

## Parameters

### Body

**Content type:** `application/json`

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `topics` | array | yes | List of topics to exit, each with their own set of legs. |

#### Topic object

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `topic_id` | integer | yes | Market (topic) identifier. |
| `legs` | array | yes | One or more legs to exit within this topic. |

#### Leg object

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `option_id` | integer | yes | Identifier of the option position to exit. |
| `contracts` | integer | yes | Number of contracts to exit for this leg. |

## Request

```bash
curl -X 'POST' \
  'https://main.bpmapi.io/api/v1/nmarket/exit-multi-topic-multi-leg' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer <token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "topics": [
      {
        "topic_id": 1944,
        "legs": [
          {
            "option_id": 101,
            "contracts": 10
          }
        ]
      },
      {
        "topic_id": 1945,
        "legs": [
          {
            "option_id": 204,
            "contracts": 5
          },
          {
            "option_id": 205,
            "contracts": 3
          }
        ]
      }
    ]
  }'
```

## Responses

### `200 OK`

**Content type:** `application/json`

**Schema**

```json
{
  "results": [
    {
      "commission_millisats": 0,
      "error": "string",
      "topic_id": 0,
      "total_proceeds_millisats": 0,
      "trade_id": "string"
    }
  ],
  "topics_exited": 0,
  "topics_failed": 0,
  "topics_requested": 0,
  "total_commission_millisats": 0,
  "total_proceeds_millisats": 0
}
```

**Fields**

| Field | Type | Description |
| --- | --- | --- |
| `topics_requested` | integer | Total number of topics submitted in the request. |
| `topics_exited` | integer | Number of topics successfully exited. |
| `topics_failed` | integer | Number of topics that failed to exit. |
| `total_proceeds_millisats` | integer | Total proceeds across all exited positions, in millisats. |
| `total_commission_millisats` | integer | Total commission charged across all exits, in millisats. |
| `results` | array | Per-topic breakdown of exit results. |

#### Result object

| Field | Type | Description |
| --- | --- | --- |
| `topic_id` | integer | Market identifier for this exit result. |
| `trade_id` | string | Unique identifier of the executed exit trade. |
| `total_proceeds_millisats` | integer | Proceeds from this position exit, in millisats. |
| `commission_millisats` | integer | Commission charged for this exit, in millisats. |
| `error` | string | Error message if this topic failed to exit; empty string on success. |
