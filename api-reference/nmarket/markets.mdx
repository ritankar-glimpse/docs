<Info>
  **Endpoint:** `GET /api/v1/nmarket/batches/{batch_id}/markets`\
  **Auth:** User JWT (raw token in `Authorization` header, no `Bearer ` prefix)
</Info>

Returns all markets for the given batch, including both active and resolved markets.

## Parameters

### Path

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `batch_id` | string (UUID) | yes | Batch to fetch markets for. |

## Request

```bash
curl -X 'GET' \
  'https://main.bpmapi.io/api/v1/nmarket/batches/c6dd0be7-b9b8-4a8a-b735-623660c38982/markets' \
  -H 'accept: application/json' \
  -H 'Authorization: <USER_JWT>'
```
