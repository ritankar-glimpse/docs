---
title: "Nmarket"
description: "Newer batch-aware market engine: markets, trades, portfolio, multi-leg bets"
---

N-Market is the newer, parallel market engine running alongside the legacy [Markets & Topics](/api-reference/markets-topics) routes. All paths are prefixed `/api/v1/nmarket/...`.

## Markets & batches (Public)

| Method | Path | Purpose |
| --- | --- | --- |
| GET | `/api/v1/nmarket/markets` | List markets. |
| GET | `/api/v1/nmarket/markets/:topic_id/quotes` | Quotes for a market's topic. |
| GET | `/api/v1/nmarket/markets/:topic_id/volume` | Trading volume by option for a topic. |
| GET | `/api/v1/nmarket/batches` | List market batches. |
| GET | `/api/v1/nmarket/batches/:batch_id/markets` | All markets in a batch. |
| GET | `/api/v1/nmarket/batches/:batch_id/active-markets` | Active markets in a batch. |
| GET | `/api/v1/nmarket/v2/batches/:batch_id/active-markets` | Active markets by batch, v2 paginated. |
| GET | `/api/v1/nmarket/v2/batches/:batch_id/resolved-markets` | Resolved markets by batch, v2 paginated. |
| GET | `/api/v1/nmarket/ended-by-batch` | Ended markets by batch, paginated. |
| GET | `/api/v1/nmarket/ended-past-168h` | Markets ended in the last 168h (admin view). |
| GET | `/api/v1/nmarket/resolved-past-168h` | Markets resolved in the last 168h. |

## Trading

| Method | Path | Auth | Purpose |
| --- | --- | --- | --- |
| POST | `/api/v1/nmarket/trades/estimate` | Public | Estimate cost / outcome of a trade before executing. |
| POST | `/api/v1/nmarket/trades` | User JWT \+ Eligibility | Execute a trade (n-market engine). |
| POST | `/api/v1/nmarket/enter-multi-topic-multi-leg` | User JWT \+ Eligibility | Enter a multi-leg bet spanning multiple topics. |
| POST | `/api/v1/nmarket/exit-multi-topic-multi-leg` | User JWT | Exit a multi-leg, multi-topic bet. |
| POST | `/api/v1/nmarket/exit-batch` | User JWT | Exit all positions in a batch. |
| POST | `/api/v1/nmarket/exit-consolidated` | User JWT | Exit one consolidated position. |
| POST | `/api/v1/nmarket/exit-consolidated-multi` | User JWT | Exit multiple consolidated positions. |
